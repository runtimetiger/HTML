Text Formating Tags

Bold text uses <b> or <strong> tag

Italic text uses <i> or <em> tag

Underline text uses <u> or <ins> tag

Strike through text uses <s> or <del> tag

Mark text uses <mark> tag

Small text uses <small> tag

Superscript text uses <sup> tag

Subscript text uses <sub> tag

Quotation
text uses <blockquote> tag
